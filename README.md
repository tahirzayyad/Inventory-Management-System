# Inventory Management System

# Overview
A Java-based application for managing products, suppliers, and stock levels in a business.  
It allows users to add, update, and track inventory in real time.

# Features
- Add, edit, and delete inventory items  
- Supplier management  
- Low-stock alerts  
- Simple login system

# Technologies Used
- Java 17  
- SQLite (for local database)  
- Maven (for build automation)

# Project Structure
- `src/main/java` – Source code  
- `src/test/java` – Test cases  
- `docs/` – Documentation and design diagrams  

# How to Run
1. Clone this repository  
2. Open in IntelliJ or VS Code  
3. Build using Maven or Gradle  
4. Run `Main.java`
